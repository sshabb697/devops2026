# Day 4 — Azure Boards

Today you'll learn to **plan and track work** with Azure Boards. You'll meet the work item hierarchy (Epics → Features → User Stories → Tasks/Bugs), organize work into backlogs and sprints, visualize flow on Kanban boards, and report progress with queries and dashboards.

## Learning Objectives

By the end of today you will be able to:

- Navigate Azure Boards and understand its building blocks.
- Create and link **work items**: Epics, Features, User Stories, Tasks, Bugs.
- Manage **backlogs** and plan **sprints** with capacity.
- Use the **Kanban board** and **Taskboard** to visualize and update work.
- Build **queries** and **dashboards** for reporting.
- Use the project **Wiki** and configure **notifications**.
