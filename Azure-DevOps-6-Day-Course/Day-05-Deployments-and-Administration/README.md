# Day 5 — Deployments & Administration

Today you'll learn how to **deploy safely** and **administer** Azure DevOps. You'll work with environments, deployment jobs, service connections, the Library, approvals, and checks — then cover the security and administrative settings that keep an organization healthy and compliant.

## Learning Objectives

By the end of today you will be able to:

- Define **environments** and **deployment jobs** with strategies.
- Create **service connections** to deploy to external services.
- Use the **Library** (variable groups, secure files) for shared config/secrets.
- Gate deployments with **approvals** and **checks**.
- Secure **repositories** and **pipelines**.
- Administer **agent pools**, **organization** and **project settings**.
- Extend Azure DevOps with **extensions** and review **audit logs**.
