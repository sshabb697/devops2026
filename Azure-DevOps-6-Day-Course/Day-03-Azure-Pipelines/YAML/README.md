# Sample Pipeline YAML

Ready-to-use Azure Pipelines YAML files referenced by the Day 3 lessons. Copy one into your repository as `azure-pipelines.yml` and adapt it.

| File | Description |
| ---- | ----------- |
| [`ci-dotnet.yml`](./ci-dotnet.yml) | CI build for a .NET project |
| [`ci-node.yml`](./ci-node.yml) | CI build for a Node.js project |
| [`ci-python.yml`](./ci-python.yml) | CI build for a Python project |
| [`ci-java-maven.yml`](./ci-java-maven.yml) | CI build for a Java/Maven project |
| [`multi-stage.yml`](./multi-stage.yml) | Multi-stage build → Dev → Prod pipeline |
| [`templates/build-steps.yml`](./templates/build-steps.yml) | Reusable steps template |

> See the [Sample-Applications](../../Sample-Applications/) folder for matching demo apps.
