# Day 1 — Azure DevOps Fundamentals

Welcome to Day 1! Today builds the conceptual foundation for the rest of the course. You will learn what DevOps *is* (and isn't), how software flows through the DevOps lifecycle, and how the Azure DevOps platform is structured so you can navigate it confidently.

## Learning Objectives

By the end of today you will be able to:

- Define DevOps and explain why organizations adopt it.
- Describe each phase of the DevOps lifecycle.
- Distinguish Continuous Integration, Delivery, and Deployment.
- Compare Agile and Waterfall, and explain the Scrum framework.
- Describe Azure DevOps, its architecture, and its five services.
- Create and organize Azure DevOps **organizations** and **projects**.
- Choose the right **process template**.
- Manage **users, groups, permissions**, and **access levels**.
